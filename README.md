# Fake Check-in

AppStore連結：https://itunes.apple.com/tw/app/fake-check-in/id1021216357?mt=8

功能單純的Facebook應用app，可搜尋並在任何你想要的地標打卡，並且標記好友、上傳照片等。

純屬博君一笑，請勿用於不當用途！

# 適用環境

iPhone (iOS 8.0以上)
